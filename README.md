## ✨ Features

| Feature | Description |
|--------|-------------|
| 📞 **Phone Intelligence** | Carrier detection, line type, VOIP check, location, and breach data |
| 📧 **Email Intelligence** | Deliverability, quality score, domain info, spam risk, and breach history |
| 🌐 **IP Tracker** | Geolocation, ISP, VPN/Proxy/TOR detection, abuse flags, and live map |
| 👤 **Username Recon** | Hunts a target username across 25+ platforms simultaneously |
| ⚡ **Live Streaming Results** | Username results stream in real-time as each platform is probed |
| 🎯 **Multi-Proxy Fallback** | Automatic proxy rotation for maximum reachability |
| 🛡️ **Brand Protection** | MutationObserver-based anti-tamper system |
| 🌙 **Dark UI** | Premium dark interface built on the DarkNova design system |
